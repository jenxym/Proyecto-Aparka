Ejemplo de página con Tailwind CSS

Volver a index

Para instalar los archivos necesarios (en Dev)

  1. npm init
  2. git init
  3. npm install -D parcel-bundler
  4. npm install -D postcss-cli postcss-clean (Postcss y minificador de css)
  5. npm install -D autoprefixer (añadir prefijos para navegadores antiguos)
  6. npm install -D tailwindcss
  7. Crear archivo de postcss: code .postcssrc
  8. npx tailwindcss init (crea archivo de configuración)
  9. tailwindcss intellisense plugin vscode para ver tags

Archivo .postcssrc

            {
                "plugins": {
                    "autoprefixer": true,
                    "postcss-clean": true,
                    "tailwindcss": true
                }
            }

En el arvhico index.css poner

            @tailwind base;

            @tailwind components;

            @tailwind utilities;
