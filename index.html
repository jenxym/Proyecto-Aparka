Para instalar los archivos necesarios (en Dev)

  1. npm init, git init
  2. npm install -D parcel-bundler
  3. npm install -D postcss-cli postcss-clean
  4. npm install -D autoprefixer
  5. npm install -D tailwindcss
  6. Crear archivo de postcc: code .postcssrc

Archivo postcssrc

        {
            "plugins": {
                "autoprefixer": true,
                "postcss-clean": true,
                "tailwindcss": true
            }
        }

En el arvhico index.css poner

        @tailwind base;

        @tailwind components;

        @tailwind utilities;
